# Star Wars Info App

A React app that displays information about Star Wars films using the [SWAPI](https://swapi.dev/) API.

This project is also a practice project for `React Query`.

[Live Demo](https://starwars-info-d5c7ac.netlify.app/)

## Technologies used
- `React.js`
- `React Query`
- `Vite`
